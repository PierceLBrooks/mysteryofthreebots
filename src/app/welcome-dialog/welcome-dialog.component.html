Welcome to the Mystery of the Three Bots

This simple dialog-based game is an example of using a machine-learning model inside of a web application. It is open sourced at https://github.com/google/mysteryofthreebots.

The model it uses is called Universal Sentence Encoder Lite and is available at Tensorflow Hub.

A helpful tool for creating dialog-based games is the Semantic Reactor, a Google Sheets Add-On that allows you to experiment with machine-learned language models. To learn more, go to the Google Cloud AI Workshop Page.
